# # 台股短期趨勢預測與決策輔助系統
> 結合機器學習模型與技術指標，建立具備信心門檻機制的台股短期趨勢預測系統，將模型輸出轉化為較具參考價值的操作建議。

🔗 Live Demo: https://twstockproject-l36mxwq44t8fezekxuhroc.streamlit.app/

--- 

## 📌 專案概述（Project Overview）
### 🔹專案目標：
預測**未來第 3 個交易日**收盤價相對於當前的漲跌方向，並提供簡單明確的決策建議。

### 🔹核心設計
- **信號整合（Signal Integration）**
  - 結合模型預測方向、信心值（Confidence Score）與技術面指標，作為最終判斷依據，而非單一模型輸出。
  
- **動態門檻策略（Dynamic Thresholding）**
  - 使用者可自訂信心門檻，僅在模型信心較高時輸出「看多/看空」，否則顯示「觀望」，降低 低確定性訊號 的干擾。

- **視覺化輔助**
  - 透過 Plotly 互動式 K 線圖與成交量，讓使用者能同時觀察價格走勢與模型判斷。

---

## 💡 專案亮點
- **AI 與技術分析結合**  
  將機器學習預測結果與技術指標搭配使用，作為輔助判斷工具，而非單純依賴模型輸出。

- **Decision Layer（決策層）設計**  
  透過信心門檻將預測結果轉換為「看多/看空/觀望」，讓輸出更貼近實際決策情境。

- **互動式金融視覺化**  
  使用 Plotly 建立 K 線圖與成交量圖，並整合均線與預測標記，提升資訊可讀性。

- **資料流程一致性設計**  
  統一資料處理與特徵工程流程，確保訓練與推論使用相同邏輯。

- **模型可解釋性**  
  透過 Feature Importance 顯示主要影響因子，提供基本的模型解釋能力。

## 📊 資料與特徵工程（Dataset & Feature Engineering）
### 🔹資料來源
- 使用 Yahoo Finance 歷史股價資料（OHLC + Volume），並於資料前處理階段進行格式統一與資料檢查，以確保後續分析與建模的一致性。

### 🔹特徵工程（多維度特徵）
- **趨勢類**：MA5/MA20、均線斜率、均線乖離率  
- **動能類**：KD 指標、短期報酬率（1/3/5日）  
- **價格結構**：價格與均線比值、布林通道位置  
- **成交量**：成交量比例（Volume Ratio）

### 🔹模型設計
- 模型：Random Forest、XGBoost
- 方法：時間序列切分（避免資料洩漏）
- 每支股票獨立訓練模型，並依測試表現選擇最佳模型（Random Forest 或 XGBoost）
- 表現（不同股票略有差異）：
  - 約落在 49%～55% 區間
  - 與市場隨機基準（約 50%）接近

> 模型預測表現接近隨機基準，顯示在目前特徵設定下，短期價格方向的可預測性有限，並指出僅依歷史價格資訊建模之侷限，後續可透過引入其他市場特徵進一步改善。
